# 药水效果修复总结

## 🔧 **修复的问题**

### 1. **药水效果类型名称错误** ✅
在 **1.20.1** 版本中，正确的药水效果类型是：
- **力量效果**: `PotionEffectType.INCREASE_DAMAGE` (不是 `STRENGTH`)
- **伤害抗性**: `PotionEffectType.DAMAGE_RESISTANCE`

### 2. **修复的文件**

#### **EvolutionManager.java**
```java
// 修复前
mob.addPotionEffect(new org.bukkit.potion.PotionEffect(
    PotionEffectType.STRENGTH, 
    Integer.MAX_VALUE, 1, true, false));

// 修复后
mob.addPotionEffect(new org.bukkit.potion.PotionEffect(
    PotionEffectType.INCREASE_DAMAGE, 
    Integer.MAX_VALUE, 1, true, false));
```

#### **EvolutionEffects.java**
```java
// 修复前
mob.addPotionEffect(new PotionEffect(PotionEffectType.STRENGTH, Integer.MAX_VALUE, 0));

// 修复后
mob.addPotionEffect(new PotionEffect(PotionEffectType.INCREASE_DAMAGE, Integer.MAX_VALUE, 0));
```

#### **ChallengeTowerConfig.java**
```java
// 修复前
specialEffects.add("STRENGTH");             // 力量效果

// 修复后
specialEffects.add("INCREASE_DAMAGE");      // 力量效果
```

## 📝 **1.20.1版本药水效果类型**

### 正确的药水效果类型
| 效果 | 1.20.1中的名称 | 说明 |
|------|----------------|------|
| 力量 | `INCREASE_DAMAGE` | 增加攻击伤害 |
| 伤害抗性 | `DAMAGE_RESISTANCE` | 减少受到的伤害 |
| 火焰抗性 | `FIRE_RESISTANCE` | 免疫火焰伤害 |
| 击退抗性 | `KNOCKBACK_RESISTANCE` | 减少击退效果 |
| 生命恢复 | `REGENERATION` | 持续恢复生命值 |
| 速度 | `SPEED` | 增加移动速度 |

### 常见错误
- ❌ `STRENGTH` - 在1.20.1中不存在
- ✅ `INCREASE_DAMAGE` - 正确的力量效果
- ✅ `DAMAGE_RESISTANCE` - 正确的伤害抗性
