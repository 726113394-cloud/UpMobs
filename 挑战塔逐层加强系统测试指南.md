# 挑战塔逐层加强系统测试指南

## 🏗️ 系统概述
挑战塔现在实现了完整的逐层加强系统，怪物会根据层数获得多种增强效果，包括属性增强、特殊效果、进化系统等。

## 📊 增强阶段划分

### 1. 基础层（1-5层）
- **属性小幅增强**：血量、伤害、速度、护甲逐步提升
- **基础怪物**：普通怪物，无特殊效果
- **怪物数量**：10-34只（每层+6只）

### 2. 进阶层（6-10层）
- **属性中等增强**：属性倍率继续提升
- **进化怪物**：开始出现进化怪物（几率逐层增加）
- **特殊效果**：开始出现特殊效果（几率逐层增加）
- **怪物数量**：40-64只

### 3. 精英层（11-15层）
- **属性大幅增强**：属性倍率显著提升
- **第四阶段怪物**：开始出现第四阶段怪物（几率逐层增加）
- **高进化率**：进化怪物几率达到15-30%
- **怪物数量**：70-94只

### 4. 大师层（16-18层）
- **属性极限增强**：属性倍率达到最大值
- **高特殊效果率**：特殊效果几率达到35-50%
- **高第四阶段率**：第四阶段怪物几率达到8-15%
- **怪物数量**：100-118只

## ⚙️ 增强因素详解

### 1. 属性增强（逐层线性增加）
- **血量倍率**：1.0x → 10.0x（每层+0.5x）
- **伤害倍率**：1.0x → 8.0x（每层+0.4x）
- **速度倍率**：1.0x → 3.0x（每层+0.1x）
- **护甲倍率**：1.0x → 5.0x（每层+0.2x）
- **击退抗性**：每层增加5%，最大80%

### 2. 特殊效果系统
- **开始层数**：第5层开始
- **效果几率**：每层增加3%，最大50%
- **效果类型**：
  - `FIRE_RESISTANCE`：火焰抗性
  - `KNOCKBACK_RESISTANCE`：击退抗性
  - `REGENERATION`：生命恢复
  - `STRENGTH`：力量效果
- **效果等级**：每3层增加1级，最大2级

### 3. 进化系统
- **开始层数**：第6层开始
- **进化几率**：每层增加1.7%，最大30%
- **进化阶段**：根据层数决定（1-3阶段）
- **进化效果**：应用UpMobs的进化系统

### 4. 第四阶段怪物
- **开始层数**：第11层开始
- **出现几率**：每层增加0.8%，最大15%
- **额外增强**：
  - 额外50%血量
  - 额外30%伤害
  - 发光效果
  - 精英标签

### 5. 升格怪物
- **出现几率**：每层增加5%，最大80%
- **增强效果**：应用UpMobs的升格系统

## 🎮 测试步骤

### 步骤1：环境准备
1. 确保UpMobs插件已安装并启用
2. 确保挑战塔系统已配置并可用
3. 准备测试玩家账号

### 步骤2：基础层测试（1-5层）
1. 进入挑战塔第1层
2. 观察怪物属性是否小幅增强
3. 检查怪物标签：`tower_difficulty_basic`
4. 验证怪物数量是否正确（10只基础，每层+6只）
5. 确认没有特殊效果和进化怪物

### 步骤3：进阶层测试（6-10层）
1. 进入挑战塔第6层
2. 观察怪物属性是否中等增强
3. 检查怪物标签：`tower_difficulty_advanced`
4. 验证是否出现进化怪物（标签：`tower_evolution_stage_1`）
5. 验证是否出现特殊效果（标签：`tower_effect_*`）
6. 检查进化几率是否正确（第6层约1.7%）

### 步骤4：精英层测试（11-15层）
1. 进入挑战塔第11层
2. 观察怪物属性是否大幅增强
3. 检查怪物标签：`tower_difficulty_elite`
4. 验证是否出现第四阶段怪物（标签：`tower_stage4`）
5. 检查第四阶段怪物是否有发光效果
6. 验证进化几率是否增加（第11层约8.5%）

### 步骤5：大师层测试（16-18层）
1. 进入挑战塔第16层
2. 观察怪物属性是否极限增强
3. 检查怪物标签：`tower_difficulty_master`
4. 验证特殊效果几率是否达到35%以上
5. 验证第四阶段怪物几率是否达到8%以上
6. 检查击退抗性是否接近最大值

### 步骤6：增强信息显示测试
1. 进入每一层的第一波
2. 验证是否显示层数增强信息
3. 检查增强信息是否包含：
   - 属性倍率
   - 特殊效果几率
   - 进化怪物几率
   - 第四阶段怪物几率
   - 升格怪物几率

## 🔧 配置调整

### 1. 调整属性增强速率
```yaml
level_scaling:
  health_multiplier_per_level: 0.3    # 降低血量增强速率
  damage_multiplier_per_level: 0.25   # 降低伤害增强速率
  max_health_multiplier: 5.0          # 降低最大血量倍率
  max_damage_multiplier: 4.0          # 降低最大伤害倍率
```

### 2. 调整特殊效果
```yaml
level_scaling:
  min_level_for_effects: 3            # 提前特殊效果出现层数
  effect_chance_per_level: 0.05       # 提高特殊效果几率
  max_effect_chance: 0.7              # 提高最大特殊效果几率
  special_effects:                    # 添加更多效果类型
    - FIRE_RESISTANCE
    - KNOCKBACK_RESISTANCE
    - REGENERATION
    - STRENGTH
    - SPEED
    - RESISTANCE
```

### 3. 调整进化系统
```yaml
level_scaling:
  min_level_for_evolution: 3          # 提前进化怪物出现层数
  evolution_chance_per_level: 0.03    # 提高进化几率
  max_evolution_chance: 0.5           # 提高最大进化几率
```

### 4. 调整第四阶段怪物
```yaml
level_scaling:
  min_level_for_stage4: 8             # 提前第四阶段怪物出现层数
  stage4_chance_per_level: 0.015      # 提高第四阶段怪物几率
  max_stage4_chance: 0.25             # 提高最大第四阶段怪物几率
```

## 🐛 故障排除

### 问题1：怪物属性没有增强
1. 检查配置文件是否正确加载
2. 检查 `ChallengeTowerMonsterEnhancer` 是否正确初始化
3. 检查属性增强方法是否被调用
4. 验证层数参数是否正确传递

### 问题2：特殊效果没有出现
1. 检查是否达到最小层数（默认第5层）
2. 检查特殊效果几率计算是否正确
3. 检查效果类型名称是否正确
4. 验证随机数生成是否正常工作

### 问题3：进化怪物没有出现
1. 检查是否达到最小层数（默认第6层）
2. 检查进化几率计算是否正确
3. 检查进化管理器是否正确初始化
4. 验证进化标签是否正确添加

### 问题4：第四阶段怪物没有出现
1. 检查是否达到最小层数（默认第11层）
2. 检查第四阶段怪物几率计算是否正确
3. 检查发光效果是否正确应用
4. 验证第四阶段标签是否正确添加

### 问题5：增强信息没有显示
1. 检查是否在第一波
2. 检查增强信息生成方法是否正确
3. 检查广播消息方法是否正常工作
4. 验证字符串格式化是否正确

## 📈 性能考虑

### 1. 计算复杂度
- 属性增强：O(1) 复杂度
- 特殊效果：O(1) 复杂度
- 进化系统：O(1) 复杂度
- 总体复杂度：O(n)（n为怪物数量）

### 2. 内存使用
- 每个怪物增加少量标签（约5-10个）
- 增强器对象为单例，内存占用小
- 配置对象缓存，避免重复加载

### 3. 优化建议
- 批量处理怪物增强（如果怪物数量很多）
- 缓存属性计算（如果层数固定）
- 异步处理特殊效果应用
