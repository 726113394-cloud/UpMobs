```java
public double getDifficultyMultiplierForTime(int elapsedMinutes) {
    double multiplier = 1.0 + (difficultyIncreasePerMinute * elapsedMinutes);
    return Math.min(multiplier, maxDifficultyMultiplier);
}

// 奖励倍率计算  
public double getRewardMultiplierForTime(int elapsedMinutes) {
    return Math.pow(rewardMultiplierPerMinute, elapsedMinutes);
}
```

### 4. 怪物生成
```java
// 创建第四阶段怪物
private LivingEntity createStage4Monster(Location location, String monsterTypeName) {
    // 解析怪物类型
    String baseTypeName = monsterTypeName.replace("STAGE4_", "");
    EntityType entityType = EntityType.valueOf(baseTypeName);
    
    // 生成怪物
    LivingEntity monster = (LivingEntity) location.getWorld().spawnEntity(location, entityType);
    
    // 应用第四阶段增强
    mobManager.upgradeMob(monster);
    monster.addScoreboardTag("stage4");
    
    // 第四阶段属性增强
    AttributeInstance maxHealth = monster.getAttribute(Attribute.GENERIC_MAX_HEALTH);
    if (maxHealth != null) {
        maxHealth.setBaseValue(maxHealth.getBaseValue() * 4.0);
        monster.setHealth(maxHealth.getValue());
    }
    
    return monster;
}
```

## 🚀 部署指南

### 1. 环境要求
- **Minecraft版本**：1.16+
- **Java版本**：Java 8+
- **依赖插件**：Vault（经济奖励）
- **内存要求**：建议2GB+服务器内存

### 2. 安装步骤
1. 下载插件JAR文件
2. 放入plugins文件夹
3. 启动服务器
4. 配置通天层参数
5. 创建挑战塔区域
6. 测试通天层功能

### 3. 配置建议
```yaml

sky_tower:
  monster:
    base_count: 10-15           # 根据服务器性能调整
    refresh_delay_seconds: 3-5  # 避免刷新过快
  
  time_limit:
    minutes: 10-15              # 合理的挑战时间
  
  difficulty:
    increase_per_minute: 0.1-0.15 # 适中的难度增长
  
  rewards:
```
