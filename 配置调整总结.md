# 配置调整总结

## 🎯 **问题解答**

### 1. **为什么怪物太强？**
**原因**：多重增强叠加导致怪物属性爆炸
```
最终属性 = 基础属性 × 全局倍率 × 升格增强 × 进化增强
```

### 2. **为什么速度变态？**
**原因**：速度增强在Minecraft中效果显著
- 原版僵尸速度：0.2
- 调整前：0.2 × 1.5 × 5 × 1.4 = 2.1（10.5倍！）
- 调整后：0.2 × 1.1 × 1.5 × 1.3 = 0.43（2.15倍）

### 3. **为什么都是第四阶段？**
**原因**：进化条件太容易达成
- 调整前：3次攻击，10点伤害
- 调整后：5次攻击，25点伤害，血量低于50%

### 4. **[升格]标签 vs 第四阶段**
- **[升格]**：自然刷新时随机强化（现在20%增强）
- **第四阶段**：吸血进化最高阶段（现在最多3阶段）
- **调整后**：[升格]不再比第四阶段更恐怖

## 🔧 **具体调整**

### 1. **全局倍率**（最关键）
- **从**: 5.0倍 → **到**: 1.5倍
- **效果**: 整体强度降低70%

### 2. **自然刷新升格**
- **生成几率**: 30% → 10%（更稀有）
- **增强幅度**: 50% → 20%（更合理）

### 3. **吸血进化系统**
- **攻击次数**: 3次 → 5次（更难进化）
- **伤害要求**: 10点 → 25点（更高要求）
- **血量阈值**: 70% → 50%（更低血量才进化）
- **最大阶段**: 4阶段 → 3阶段（降低上限）

### 4. **怪物基础属性**
#### 僵尸：
- **血量**: 500% → 200%（5倍→2倍）
- **速度**: 150% → 110%（1.5倍→1.1倍）
- **护甲**: 300% → 150%（3倍→1.5倍）
- **伤害**: 400% → 200%（4倍→2倍）
- **击退抗性**: 50% → 20%（更容易击退）

#### 骷髅：
- **血量**: 300% → 180%（3倍→1.8倍）
- **速度**: 120% → 105%（1.2倍→1.05倍）
- **伤害**: 350% → 180%（3.5倍→1.8倍）

- 需要策略应对[升格]怪物
- 团队合作应对进化怪物

### 怪物强度梯度：
1. **普通怪物**：1.5倍增强（有挑战但可应对）

3. **进化怪物**：逐步增强，最多3阶段（需要团队）

- **[升格]战斗**：1-2分钟
- **进化怪物**：2-3分钟（团队）
